Available Commands

Usage: ctnr [options] [command]
Available subcommands: login, logout, run, list, attach
login - Authenticate with ctnr.io
logout - Sign out from ctnr.io
run [options] - Deploy and run your containers
list - List your deployments
attach [options] - Connect to a running container
help [command] - Display help for command
